# UserRegisterAPI
Este projeto é uma API construída utilizando Java, Spring Boot e MySQL como banco de dados.

## API endpoints:
Esta Api oferece os seguintes endpoints:

*GET USERS*
```markdown
GET /users - Retorna a lista com todos os usuários.
```
